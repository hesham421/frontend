Status and label badge for table cells, record states, counts.

```jsx
<Badge tone="success">Active</Badge>
<Badge tone="warning" variant="soft" icon="ti ti-clock">Pending</Badge>
<Badge tone="info" variant="solid">Posted</Badge>
```

Tones: `neutral` `info` `success` `warning` `danger` `accent`. Variants: `soft` (default), `solid`, `outline`.
