KPI / metric tile for dashboards. Value uses the mono face; a left accent rail carries brand color.

```jsx
<Stat label="Open Journals" value="1,284" delta="+4.2%" deltaTone="success" accent="blue" icon="ti ti-book" />
```

`accent` colors the rail (`blue` `teal` `slate` `amber` `green` `red`). `delta` + `deltaTone` show trend.
