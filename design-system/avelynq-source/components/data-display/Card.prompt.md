The foundational surface — wraps content, forms, tables, and dashboard panels.

```jsx
<Card title="Chart of Accounts" subtitle="General Ledger" actions={<Button size="sm">Add</Button>}>
  …content…
</Card>
```

Props: `title`, `subtitle`, `actions` (header right), `header` (custom header override), `footer`, `padding` (`none` for tables), `hover` (lift on hover for clickable cards).
