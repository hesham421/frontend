The AVELYNQ action button — confident, structured, enterprise. Use for any primary or secondary action.

```jsx
<Button variant="primary" iconLeft={<i className="ti ti-plus" />}>Add Account</Button>
<Button variant="secondary">Cancel</Button>
<Button variant="accent">Approve</Button>
<Button variant="danger" size="sm">Delete</Button>
```

Variants: `primary` (main blue action), `accent` (teal — growth/positive), `secondary` (neutral outline), `ghost` (low emphasis), `danger` (destructive red), `inverse` (on navy surfaces). Sizes: `sm` `md` `lg`. Props: `loading`, `disabled`, `block`, `iconLeft`, `iconRight`.
