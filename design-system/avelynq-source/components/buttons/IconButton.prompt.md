Square icon-only button for toolbars, table row actions, and topbar utilities.

```jsx
<IconButton icon="ti ti-refresh" label="Refresh" variant="outline" />
<IconButton icon="ti ti-dots-vertical" label="More" />
```

Variants: `ghost` (default, toolbar), `outline`, `solid` (brand), `subtle`. Sizes `sm` `md` `lg`. `icon` accepts a Tabler/FontAwesome class string or any React node.
